# Практика

#### Инструкция по запуску и работе с докером

---
1. Скачайте и установите Docker Desktop для вашей ОС (Windows/Mac/Linux) с официального сайта.
2. Запустите Docker Desktop и убедитесь, что он работает. В левом нижнем углу программы должен гореть зеленый статус Engine running (иконка кита в трее возле часов должна перестать мигать).
3. Установите в свой VS Code официальное расширение Dev Containers от Microsoft (ищите его на вкладке расширений слева).
4. Клонируйте наш репозиторий с GitHub на свой компьютер и откройте папку проекта в VS Code. открывайте именно корневую папку проекта
5. Потом Нажать `Ctrl + Shift + P`
6. Вводите `Dev Containers: Reopen and ...` (и что-то дальше не помню). Ждете загрузку и все готово.
